Frequently Asked Questions

Quick answers to common questions about the testnet.

What is a testnet?

A testnet is a public testing environment for the blockchain. Tokens are for experimentation only and have no monetary value.

How do I get testnet ZION?

Mine on the testnet using supported algorithms or watch for a faucet announcement in the repository. See Mining Guide.

Does testnet ZION have value?

No. Testnet coins are free and valueless, used purely for development and community testing.

When is mainnet?

There is no fixed date. Progress depends on stability, security audits, and community feedback. Track updates in releases.

Where can I report bugs or request features?

Open an issue on GitHub Issues.

What is consciousness mining?

An experimental mechanism that multiplies mining rewards based on spiritual development levels. See Consciousness Levels for details.

Can I run a pool?

Yes! See the Pool Setup Guide for technical requirements and configuration.

Which mining algorithm should I use?

RandomX is CPU-friendly, Autolykos v2 is GPU-optimized, and Cosmic Harmony is experimental. Choose based on your hardware. See Mining Guide.
