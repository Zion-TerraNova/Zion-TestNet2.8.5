# FAQ

Answers to common questions about ZION TestNet.

## How do I get testnet tokens?

Use the community faucet or request in the Discord/Telegram channel. Never buy testnet tokens.

## Which algorithms are supported?

See the [Mining Guide](/docs/mining-guide) for the current list and tuning tips.

## Is solo mining supported?

Yes. You can mine against your local node if your miner supports the protocol.
