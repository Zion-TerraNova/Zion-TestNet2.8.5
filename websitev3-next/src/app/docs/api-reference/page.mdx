# API Reference

JSON-RPC and WebSocket endpoints available on ZION TestNet.

## JSON-RPC

Endpoint:

```
https://rpc.testnet.zion.example
```

### Example: getBlockByNumber

```json
{
  "jsonrpc": "2.0",
  "method": "getBlockByNumber",
  "params": ["latest", false],
  "id": 1
}
```

## WebSocket

Subscribe to new blocks and events via WebSocket.

```
wss://ws.testnet.zion.example
```

### Subscriptions

- newHeads
- logs
