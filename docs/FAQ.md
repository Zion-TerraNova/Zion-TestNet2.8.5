# ❓ FAQ – ZION Testnet v2.8.5

### Mají testnet mince hodnotu?
Ne. Jsou určeny pouze pro testování.

### Proč nejsou veřejné zdrojové kódy?
Bezpečnost premine (15.78B ZION) a genesis logiky – distribuce je binární.

### Jak ověřím binárky?
Pomocí SHA256/PGP viz Release Notes.

### Kde nahlásit bugy?
Na GitHub Issues ve veřejném repozitáři.
