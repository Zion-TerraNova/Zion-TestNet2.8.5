# Code of Conduct – ZION Community

## Naše závazky

Vytváříme otevřené a přátelské prostředí pro všechny přispěvatele.

## Standardy chování

✅ **Podporujeme:**
- Respektující a konstruktivní komunikaci
- Přijetí různých názorů a zkušeností
- Poskytování a přijímání konstruktivní kritiky
- Zaměření na co je nejlepší pro komunitu

❌ **Nepřípustné:**
- Urážlivé nebo ponižující komentáře
- Trolling, spam nebo nekonstruktivní kritika
- Zveřejňování soukromých informací bez svolení
- Jiné chování neprofesionální povahy

## Naše odpovědnost

Správci projektu mají právo odstranit, upravit nebo odmítnout příspěvky, které nerespektují tento Code of Conduct.

## Rozsah

Tento Code of Conduct se vztahuje na všechny prostory projektu (GitHub, Discord, email).

## Vynucování

Případy nepřípustného chování hlaste na: admin@zionterranova.com
