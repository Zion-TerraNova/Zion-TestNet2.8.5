# Security

Shrnutí bezpečnostní architektury ZION 2.8.5. Kompletní text je dostupný ve složce `content/whitepaper/`.

## Konsenzus

- Hybridní RandomX + reputační multiplikátor.
- Detekce double-signing, slashování multiplikátoru i odměn.

## Mosty a interoperabilita

- Rainbow Bridge s dvoufázovým potvrzením.
- Watchtower uzly sledují anomálie, reportují do incident kanálu.

## Nouzové postupy

- Emergency Council může pozastavit mosty.
- Incidenty se hlásí na `security@zion.earth` (PGP 0x3A1C56E19F428B75).

## Audit

- SBOM a binární podpisy ke každému release.
